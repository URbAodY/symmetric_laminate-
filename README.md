# symmetric_laminate
这是一个用于计算复合材料对称层合板面内刚度和弯曲刚度的 Fortran 程序。由gfortran编译器编译。

## 程序功能

### 主要功能
1. **面内刚度计算**
   - 计算对称层合板的正则化面内刚度系数 (A11, A22, A12, A66, A16, A26)
   - 计算正则化面内柔度系数 (a11_, a22_, a12_, a66_, a16_, a26_)
   - 计算面内拉伸弹性模量Ex, 面内压缩弹性模量Ey, 面内剪切弹性模量Gxy, Ex/Gxy, Ey/Gxy, Ex/Ey, Gxy/Ey的值

2. **弯曲刚度计算**
   - 计算对称层合板的正则化弯曲刚度系数 (D11, D22, D12, D16, D26, D66)
   - 支持所有可能的铺层排列组合计算

3. **优化排序**
   - 支持多种优化目标：Ex, Ey, Gxy 及其比值
   - 按指定优化目标对铺层方案进行排序

## 输入格式

### 材料参数输入
程序支持两种输入方式：

1. **文件输入** 

2. **手动输入**
- 按提示依次输入每种材料的参数

### 铺层设计输入
- 使用4位数字编码表示铺层顺序
- 例如：`4399` 表示 `[0_4/90_1/45_9/-45_9]s` 层合板
- 格式：`[0度铺层数/90度铺层数/45度铺层数/-45度铺层数]s`

## 输出文件

程序会生成以下输出文件：

- `all_output.txt` - 所有铺层设计的完整计算结果
- `output.txt` - 优化后的最佳结果(未完成)
- `all_D_output.txt` - 所有弯曲刚度计算结果
- `D_output.txt` - 优化后的最佳结果(未完成)
